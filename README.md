## Uso del proyecto:

Se necesita el CLI de Ionic con soporte para aplicaciones de la version 2:

```bash
$ npm install -g ionic
```

y despues:

```bash
$ ionic start dondevoy https://github.com/nachomieres/dondevas --v2
```
